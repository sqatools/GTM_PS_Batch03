1.Python:
2.Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.
3.It is used for:
4.web development (server-side),
5.software development,
6.mathematics,
7.system scripting.
8.What can Python do?
9.Python can be used on a server to create web applications.
10.Python can be used alongside software to create workflows.
11.Python can connect to database systems. It can also read and modify files.
12.Python can be used to handle big data and perform complex mathematics.
13.Python can be used for rapid prototyping, or for production-ready software development.
